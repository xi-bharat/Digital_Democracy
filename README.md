# 🗳️ Digital Democracy

A secure Android-based Online Voting Application built using **Java**, **Android Studio**, and **Firebase**. Digital Democracy simplifies the election process by enabling authenticated users to cast their votes digitally while ensuring transparency, fairness, and the principle of **one voter, one vote**.

---

## 📱 Features

- 🔐 Secure User Authentication
- 👤 Voter Registration & Login
- 🏛️ Candidate Management
- 🗳️ One Vote Per User
- 📊 Real-time Vote Counting
- 📈 Live Election Results
- ☁️ Firebase Realtime Database Integration
- 📱 Clean & User-Friendly Android UI

## 🚀 Getting Started

### Prerequisites

- Android Studio
- JDK 8+
- Firebase Project
- Android Device or Emulator

### Installation

1. Clone the repository

```bash
git clone https://github.com/xi-bharat/Digital_Democracy.git
```

2. Open the project in **Android Studio**

3. Connect Firebase to the project.

4. Sync Gradle.

5. Run the application on an emulator or Android device.
